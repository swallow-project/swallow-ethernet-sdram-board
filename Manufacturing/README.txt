Description of files included in Swallow-ethernet-FAB.zip:

./README.txt    This File

./gerbers/   This directory contains the following gerber files for the design.

Swallow-ethernet-TOP.gtl	Top Copper Layer 	(Layer 1)
Swallow-ethernet-BTM.gbl	Bottom Copper Layer 	(Layer 2)
Swallow-ethernet-SMT.gts	Top Solder Mask
Swallow-ethernet-SMB.gbs	Bottom Solder Mask
Swallow-ethernet-SPT.gtp	Top Solder Paste
Swallow-ethernet-SST.gto	Top Silk Screen
Swallow-ethernet-COM.gbr	Comments Layer (for information only)
Swallow-ethernet-DRL.pho	Gerber format Drill File (plated through holes)
Swallow-ethernet-NPTH-DRL.pho   Gerber format Drill File (non-plated through holes)


./drills

Swallow-ethernet-DRL.drl	Excellon format drill file, plated holes - Decimal, absolute co-ordinates. Hole size is finished size.
Swallow-ethernet-NPTH-DRL.drl   Excellon format drill file, non-plated holes (mechanical) - Decimal, absolute co-ordinates. Hole size is finished size.
Swallow-ethernet-DRL-REPORT.rpt	Text drill size and number summaryREP


./bom

Swallow-ethernet-BOM.xls	Excel format Bill of Materials


./xy/			XY Coponent poisition data (components are placed on front side only)

Swallow-ethernet-XY-Front.csv	XY Component Position Data (Front side) (Comma seperated format)
Swallow-ethernet-XY-Front.xls	XY Component Position Data (Front side) (Excel format)


./netlist/      This directory contains the netlist files for equivalence checking and bare board test.

Swallow-ethernet-Netlist.net	Netlist (KiCad format)

./Schematic
Swallow-ethernet-Schematic.pdf	Schematic (.pdf format)
